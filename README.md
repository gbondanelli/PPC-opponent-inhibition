# PPC opponent inhibition models
Supplementary code for the paper "Synaptic wiring motifs in posterior parietal cortex support decision-making" by A. T. Kuan, G. Bondanelli, L. N. Driscoll, J. Han, M. Kim, D. G. C. Hildebrand, B. J. Graham, L. A. Thomas, S. Panzeri, C. D. Harvey, W. C. A. Lee 

The repository contains the code (jupyter notebooks) used to generate the panels of Fig.4 and Exr. Data Fig.4.

- The notebook gen_fig4_panels_meanfield.ipynb generates panels relative to the linear rate model (Fig.4a-d and Ext. Data Fig. 4)
- the notebook gen_fig4_panels_RNN.ipynb generates panels relative to the recurrent neural network model (Fig.4e-i),
- the modules folder contains the modules used to in the notebooks,
- the data folder contains the data for the RNN analyses.
